Reactive vs Preemptive Vaccination Strategies: Single and two populations

Author

Jong-Hoon Kim

Published

July 5, 2025

1 The Single Population Scenario

We evaluate the expected total cost of two strategies (preemptive vs reactive vaccination) for a single population at risk of an outbreak.

1.1 Parameters

Let:

  • \(D\): The full cost of an outbreak (e.g., hospital burden, DALYs, economic cost)
  • \(C\): The cost of deploying a vaccination campaign
  • \(p\): The probability of an outbreak (\(0 < p < 1\))
  • \(r\): The relative effectiveness of a reactive vaccination campaign, where \(0 < r < 1\)

1.2 Strategy 1: Preemptive Vaccination

  • We vaccinate regardless of whether an outbreak happens
  • Outbreak is fully averted if it would have occurred
  • Always pay vaccination cost \(C\)
  • Expected outbreak cost is \(0\)

\[ \text{Cost}_{\text{pre}} = C \]

1.3 Strategy 2: Reactive Vaccination

  • Vaccinate only if an outbreak occurs (probability \(p\))
  • Outbreak is only partially mitigated: remaining cost is \((1 - r)D\)
  • Vaccination cost \(C\) is also incurred only when outbreak happens

\[ \text{Cost}_{\text{react}} = p \cdot (C + (1 - r)D) \]

1.4 Comparing the Two Strategies

We prefer preemptive vaccination when:

\[ C < p \cdot (C + (1 - r)D) \]

Solving for \(p\):

\[ p^* = \frac{C}{C + (1 - r)D} \]

This is the threshold outbreak probability above which preemptive vaccination is cost-saving.

1.4.1 Interpretation

  • As \(r \to 1\), reactive vaccination becomes nearly as effective as preemptive → threshold \(p^*\) increases
  • As \(C \to 0\), preemptive becomes cheap → threshold \(p^*\) decreases
  • As \(D \to \infty\), outbreaks become costly → preemptive is preferred at even small \(p\)

1.5 symbolic Derivation in Python (SymPy)

Code 
import sympy as sp

# Define symbols
C, D, p, r = sp.symbols('C D p r', positive=True)

# Define cost expressions
cost_pre = C
cost_react = p * (C + (1 - r)*D)

# Solve: when is preemptive better?
ineq = sp.Eq(cost_pre, cost_react)
p_thresh = sp.solve(ineq, p)[0]
p_thresh
C/(C - D*r + D)

Code

C/(C + D*(1 - r))

1.7 Summary

This analysis shows that preemptive vaccination is preferred when:

\[ p > \frac{C}{C + (1 - r)D} \]

This tradeoff depends jointly on: - Uncertainty (\(p\)), - Response effectiveness (\(r\)), - Cost of prevention (\(C\)), - Cost of inaction (\(D\))

2 Two-Population Scenario

We assume:

  • Two populations, each has an independent probability \(p\) of outbreak
  • Only one vaccination campaign is available (can be used preemptively or reactively for one population)
  • Key parameters:
Symbol Description
\(X\) Number of outbreaks (\(X=0,1,2\))
\(p\) Probability of outbreak in a population
\(C\) Cost of one vaccination campaign
\(D\) Cost of one unmitigated outbreak
\(r\) Relative effectiveness of reactive vaccination (\(0 < r < 1\))

2.1 Preemptive Strategy

We vaccinate one population in advance, say population A. No reactive response is used.

2.1.1 Case 0: No outbreaks (\(X = 0\))

  • Probability: \((1 - p)^2\)
  • Cost: only unnecessary vaccination of A → \(C\)

\[ \text{Cost}_{X=0,\text{pre}} = C \]

2.1.2 Case 1: One outbreak (\(X = 1\))

  • Probability: \(2p(1 - p)\)
  • Two subcases (each with prob. \(p(1 - p)\)):
    • If outbreak occurs in A (vaccinated): outbreak prevented → cost = \(C\)
    • If outbreak occurs in B (unvaccinated): outbreak occurs → cost = \(C + D\)

Average cost:

\[ \text{Cost}_{X=1,\text{pre}} = 2p(1 - p)\left(C + \frac{D}{2}\right) \]

2.1.3 Case 2: Two outbreaks (\(X = 2\))

  • Probability: \(p^2\)
  • One outbreak prevented (in vaccinated population), one unmitigated → cost = \(C + D\)

\[ \text{Cost}_{X=2,\text{pre}} = p^2(C + D) \]

2.1.4 Total expected cost for the preemptive strategy

\[ \begin{aligned} \mathbb{E}[\text{Cost}_{\text{pre}}] &= (1 - p)^2 C + 2p(1 - p)\left(C + \frac{D}{2} \right) + p^2(C + D) \\ &= C(1 - p)^2 + 2p(1 - p)C + p(1 - p)D + p^2 C + p^2 D \\ &= C[(1 - p)^2 + 2p(1 - p) + p^2] + D[p(1 - p) + p^2] \\ &= C + pD \end{aligned} \]

2.2 Reactive Strategy

We wait for outbreaks and respond to one if detected.

2.2.1 Case 0: No outbreaks (\(X = 0\))

  • Probability: \((1 - p)^2\)
  • No vaccination needed, no outbreak → cost = \(0\)

\[ \text{Cost}_{X=0,\text{react}} = 0 \]

2.2.2 Case 1: One outbreak (\(X = 1\))

  • Probability: \(2p(1 - p)\)
  • Detect and respond to outbreak:
    • Vaccination cost = \(C\)
    • Partial mitigation: residual cost = \((1 - r)D\)

\[ \text{Cost}_{X=1,\text{react}} = 2p(1 - p)\left(C + (1 - r)D\right) \]

2.2.3 Case 2: Two outbreaks (\(X = 2\))

  • Probability: \(p^2\)
  • Only one outbreak can be addressed:
    • Vaccination cost = \(C\)
    • One outbreak mitigated → \((1 - r)D\)
    • One outbreak unmitigated → \(D\)

\[ \text{Cost}_{X=2,\text{react}} = p^2\left(C + (1 - r)D + D\right) = p^2(C + (2 - r)D) \]

2.2.4 Total Expected Cost – Reactive Strategy

\[ \begin{aligned} \mathbb{E}[\text{Cost}_{\text{react}}] &= (1 - p)^2 \cdot 0 + 2p(1 - p)\left(C + (1 - r)D\right) + p^2\left(C + (2 - r)D\right) \\ &= 2p(1 - p)\left(C + (1 - r)D\right) + p^2\left(C + (2 - r)D\right) \end{aligned} \]

2.3 Symbolic Comparison

Compare:

\[ \mathbb{E}[\text{Cost}_{\text{pre}}] = C + pD \]

vs.

\[ \mathbb{E}[\text{Cost}_{\text{react}}] = (2p - p^2)C + \left[2p(1 - p)(1 - r) + p^2(2 - r)\right]D \]

Preemptive strategy is preferred when:

\[ C + pD < (2p - p^2)C + \left[2p(1 - p)(1 - r) + p^2(2 - r)\right]D \]

Code 
from sympy import symbols, Eq, solve, simplify

# Define symbols

# Set inequality: preemptive better than reactive
ineq = Eq(cost_pre, cost_react)

# Solve for threshold p
p_thresh = solve(ineq, p)

# Simplify solution
simplified_thresh = [simplify(expr) for expr in p_thresh]

# Output
for i, expr in enumerate(simplified_thresh):
    print(f"Threshold p solution {i+1}:")
    print(expr)

2.5 Multiple populations (\(n>2\)) scenario

We consider \(n\) populations where each has probability \(p\) of an outbreak. We can afford to vaccinate a fraction \(f\) of them, pre-emptively or reactively.

Let:

  • \(n\): number of populations
  • \(p\): outbreak probability
  • \(f\): fraction of populations we can vaccinate
  • \(C\): vaccination cost per population
  • \(D\): outbreak cost (if unmitigated)
  • \(r\): relative effectiveness of reactive vaccination (\(0 < r < 1\))

2.6 Strategy 1: Preemptive Vaccination

We preemptively vaccinate \(fn\) populations. If an outbreak occurs in the \((1 - f)n\) unvaccinated group, we pay \(D\).

Total expected cost:

\[ \text{Cost}_{\text{pre}} = fn \cdot C + (1 - f)np \cdot D \]

2.7 Strategy 2: Reactive Vaccination (Approximate)

We reactively vaccinate up to \(fn\) populations after outbreaks occur.

Expected number of outbreaks: \(np\)
We can respond to \(\min(np, fn)\) and leave \(\max(np - fn, 0)\) unmitigated.

Residual cost per mitigated outbreak = \((1 - r)D\)

Expected cost:

\[ \text{Cost}_{\text{react}} = \min(np, fn)(C + (1 - r)D) + \max(np - fn, 0) \cdot D \]

2.8 Symbolic Comparison using SymPy

Code 
import sympy as sp

# Define symbols
n, p, f, C, D, r = sp.symbols('n p f C D r', positive=True)

# Expressions
np_ = n * p
fn_ = f * n

# Preemptive strategy cost
cost_pre = fn_ * C + (1 - f) * np_ * D

# Reactive strategy cost (piecewise via min/max)
covered   = sp.Min(np_, fn_)
uncovered = sp.Max(np_ - fn_, 0)

cost_react = covered * (C + (1 - r) * D) + uncovered * D

# Symbolic difference: reactive - preemptive
cost_diff = sp.simplify(cost_react - cost_pre)
cost_diff
-C*f*n + D*n*p*(f - 1) + D*Max(0, n*(-f + p)) + (C - D*(r - 1))*Min(f*n, n*p)

# Exact cost computation for given parameters
compute_exact_costs <- function(n, p, C, D, r, f) {
  k <- floor(f * n)  # number of populations we can vaccinate

  probs <- dbinom(0:n, size = n, prob = p)

  # Preemptive: vaccinate k populations before outbreaks
  cost_pre <- sum(
    probs * (k * C + (1 - f) * (0:n) * D)  # Expected outbreak cost: only in unvaccinated
  )

  # Reactive: respond to up to k outbreaks after detection
  cost_react <- sum(
    probs * ifelse(
      0:n <= k,
      (0:n) * (C + (1 - r) * D),                    # All outbreaks mitigated
      k * (C + (1 - r) * D) + (0:n - k) * D         # Only k mitigated, rest unmitigated
    )
  )

  list(pre = cost_pre, react = cost_react)
}

find_threshold_p <- function(n, r, C, D, f, p_grid = seq(0.01, 0.99, by = 0.01)) {
  for (p in p_grid) {
    costs <- compute_exact_costs(n, p, C, D, r, f)
    if ((costs$pre - costs$react) < 0) {
      return(p)  # First p where preemptive is cheaper
    }
  }
  return(NA)
}

library(ggplot2)
library(dplyr)

# --- Parameters ---
r_vals <- seq(0.01, 0.99, by = 0.01)
p_vals <- seq(0.01, 0.99, by = 0.01)
D <- 1
C_vals <- c(0.3, 1, 2, 5)
f_vals <- c(0.4)  # exact overlay for f = 0.5
n <- 20            # exact enumeration for n = 2

# --- Function: exact expected cost given outbreak count ---
compute_exact_costs <- function(n, p, C, D, r, f) {
  k <- floor(f * n)
  x_vals <- 0:n
  probs <- dbinom(x_vals, size = n, prob = p)

  # Preemptive: vaccinate k units, pay outbreak cost only for (1 - f)
  cost_pre <- sum(probs * (k * C + (1 - f) * x_vals * D))

  # Reactive: cover up to k outbreaks after detection
  cost_react <- sum(probs * ifelse(
    x_vals <= k,
    x_vals * (C + (1 - r) * D),
    k * (C + (1 - r) * D) + (x_vals - k) * D
  ))

  list(pre = cost_pre, react = cost_react)
}

# --- Function: Find p* threshold where preemptive becomes better ---
find_threshold_p <- function(n, r, C, D, f, p_grid = seq(0.01, 0.99, by = 0.01)) {
  for (p in p_grid) {
    costs <- compute_exact_costs(n, p, C, D, r, f)
    if ((costs$pre - costs$react) < 0) {
      return(p)
    }
  }
  return(NA)
}

# --- Compute threshold p* for each (r, C) using exact method ---
threshold_exact <- expand.grid(C = C_vals, f = f_vals, r = r_vals) %>%

2.10 Asymptotic Limit: Preemptive vs Reactive

As the number of populations \(n \to \infty\) and vaccination capacity fraction \(f \to 0\) (i.e., resources are extremely limited), the expected costs for preemptive and reactive strategies simplify.

Let:

  • \(p\): probability of an outbreak per population,
  • \(r\): relative effectiveness of reactive vaccination,
  • \(C\): cost of one vaccination campaign,
  • \(D\): cost of one unmitigated outbreak,
  • \(fn\): number of populations we can afford to vaccinate (preemptively or reactively)

2.10.1 Preemptive Expected Cost

Total cost:

\[ \text{Cost}_{\text{pre}} = fn \cdot C + (1 - f)np \cdot D \]

As \(f \to 0\), this becomes:

\[ \text{Cost}_{\text{pre}} \approx npD \]

2.10.2 Reactive Expected Cost

Using the expression:

\[ \text{Cost}_{\text{react}} = \min(fn, np)(C + (1 - r)D) + \max(np - fn, 0) \cdot D \]

If \(fn \ll np\):

\[ \text{Cost}_{\text{react}} \approx fn(C + (1 - r)D) + (np - fn)D \]

2.10.3 Cost Difference

Compute:

\[ \begin{aligned} \Delta &= \text{Cost}_{\text{pre}} - \text{Cost}_{\text{react}} \\ &= fnC + (1 - f)npD - \left[fn(C + (1 - r)D) + (np - fn)D\right] \\ &= fnD(r - p) \end{aligned} \]

2.10.4 ✅ Result

  • If \(r < p\): preemptive is cheaper (\(\Delta < 0\))
  • If \(r > p\): reactive is cheaper (\(\Delta > 0\))

Thus, in the limit:

\[ p > r \quad \Rightarrow \quad \text{preemptive is preferred} \]

This corresponds to the decision threshold line \(p^* = r\) as \(f \to 0\) and \(n \to \infty\).

2.11 Optimizing Hybrid Vaccination Strategy

2.11.1 Problem

Given a limited vaccination budget (or coverage fraction \(f\)), how should we split our efforts between preemptive and reactive vaccination strategies to minimize total expected cost?

We define:

  • \(\alpha \in [0, 1]\): fraction of vaccination resources used preemptively
  • \(1 - \alpha\): fraction used reactively
  • \(f\): total fraction of the population that can be vaccinated
  • \(n\): total number of populations
  • \(p\): probability of outbreak in each population
  • \(C\): cost of one vaccination campaign
  • \(D\): cost of one unmitigated outbreak
  • \(r\): relative effectiveness of reactive vaccination

2.12 Expected Cost Formula

The total expected cost of a hybrid strategy that uses \(\alpha\) of the vaccination capacity preemptively is:

\[ \text{Cost}(\alpha) = \underbrace{\alpha f n C}_{\text{preemptive vaccination}} + \underbrace{\min(x, k_{\text{react}})(C + (1 - r)D)}_{\text{reactive cost}} + \underbrace{\max(x - k_{\text{react}}, 0) D}_{\text{unmitigated outbreaks}} \]

Where: - \(x = (1 - \alpha f)np\): expected number of outbreaks in unvaccinated group - \(k_{\text{react}} = (1 - \alpha)fn\): number of reactive campaigns available

We explore how the total expected cost varies with the fraction \(\alpha\) of vaccine budget used preemptively.
